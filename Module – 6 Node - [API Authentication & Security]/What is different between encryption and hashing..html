What is different between encryption and hashing. ? *Hashing Definition:- It is a process to convert information to a shorter fixed value known as the key that is used to represent the original information. Creation of file:- Generally, it tries to generate a new key for each information passed to the hash function but on rare occasions, it might generate the same key popularly known as a collision. Example:- MD5, SHA256 *encryption Definition:-It is the process to encode data securely such that only the authorized user who knows the key or password is able to retrieve the original data for everyone else it is just garbage. Creation of file:-It will always generate a new key for each information. Example:- RSA, AES and Blowfish
